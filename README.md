# Sharjeel Tours & Travels

A modern, premium, fully responsive taxi booking website built with React, Vite, and Tailwind CSS.

## Features

- **Modern Hero Section** with animated car showcase
- **Quick Booking Form** with WhatsApp integration
- **Services Section** showcasing all travel options
- **Popular Routes** with pricing
- **Why Choose Us** with animated features
- **Customer Testimonials**
- **Contact Section** with Google Maps
- **Floating Action Buttons** (WhatsApp & Call)
- **Fully Responsive** mobile-first design
- **Smooth Animations** with Framer Motion
- **SEO Optimized**
